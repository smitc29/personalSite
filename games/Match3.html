- FAQ -

How do I play?

Click on a tile to select it. Clicking on an adjacent tile will swap the selected tile with the most recently-clicked tile. HOWEVER, these tiles will only swap places if you selected 2 adjacent tiles, and swapping these tiles creates a row or column of 3 or more of a color in a row.

What do all of those buttons on the bottom of the screen do?

Each button provides a "skill" or power-up that can help you score more points if used carefully. The first button, which never expires, toggles outlines on tiles that can be moved to make a combo of 3 or more in a row. Each button is associated with 6 of the 8 possible tile colors, and matching tiles of a certain color restores the related button by about 5% per tile. The bigger your combo, the more points you can earn, and the faster your power-ups will become available. A button/power up will have a BLACK outline if it can be used. Clicking on a button toggles the skill, if it was clicked by accident.

Red tiles appear extremely frequently, and allow for easier combos. Matching Red tiles restores the "Reset Board" button. This button allows you to change the tiles on the board at random, which is useful if there aren't many available combos or you're getting stuck.

Blue tiles are fairly common, and allow for easier combos. Matching Blue tiles restores the "Freeze Board" button. This button will start a timer for 10 seconds upon being clicked. While the board is "Frozen," and the timer is ticking down, any adjacent tiles can be swapped, regardless of whether or not a combo is made. In addition, no combo will be resolved while the timer is counting down. This button takes more combos to be restored, but it can allow you to score massive points if used properly.

Orange tiles are somewhat uncommon. Matching Orange tiles restores the "Prime Tile" button. Upon clicking this button, it will change color. Clicking on a tile while this button has been activated will change every tile 2 or 4 spaces above, below, left, or right to the selected tile to the same color as the selected tile. This allows for easier matches, and greater combos. Combos do not immediately resolve after using this button.

Purple tiles are pretty rare. Matching Purple tiles restores the "Swap Tile" button. Upon clicking this button, it will change color. Clicking on a pair of tiles while this button has been activated will allow you to swap the location of two tiles, so long as it creates a matching combo of 3 or more. This skill recovers quickly.

Green tiles are kind of uncommon. Matching Green tiles restores the "Drop Tiles" button. Upon clicking this button, it will change color. Clicking on a tile while this button has been activated will briefly give the selected tile, and every tile below it, a Green background. Every tile with a green background will be dropped from the board, leaving any tiles in the same column as the selected tile to fall to the bottom of the column. While somewhat obtuse, it can be used to get out of a bind if used carefully.

Yellow tiles are very rare. Matching Yellow tiles restores the "Wild Cards" button. Upon clicking this button, three random tiles will become White/Rainbow tiles. Please see the following section for details.

White/Rainbow tiles are extraordinarily rare. These tiles can be matched with any other color as if it were that color. In other words, if two tiles of the same color are lined up against this tile, or this color is sandwiched between two tiles of the same color, it will count as a combo and score. Try to use this carefully, as they're prone to setting off combos quickly after entering the playing field.

Black/Weight tiles are fairly rare. These tiles don't restore any special buttons or skills on their own, and come with a severe drawback: they can't be moved UPWARDS. Black tiles can be moved left, right, or down only, unless a special skill or button is used, such as "Freeze Board" or "Drop Tiles." When a Black Tile enters the bottom row of the playing field, it will drop off the grid, and a new tile will emerge from the top!
