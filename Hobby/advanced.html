Advanced Projects

These projects are intended for artists who are at least somewhat comfortable with digital art, and have learned how to manipulate images to some degree. These lessons involve the production of vector graphics, specifically a file type known as an SVG file, which is vector-based, allowing it to scale smaller or larger without loss of quality.

Advanced Project 1

Open up Illustrator and just start using the pen tool to make some lines and shapes. Honestly I made this picture without doing too much thinking, save for the dimensions of the picture.

Project Details

  • SVG scalable format
  • Can be edited
  • Easy to replicate
  • Now in blue!
